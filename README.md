# Aurora-Watch
Project Aurora: The most over-engineered gimmick ever. A watch that is a compass, thermometer, step-counter, flashlight, all with a rechargable battery, and a processor 50x faster than the Apollo Guidance Computer.

## What is it?

It is a custom designed printed circuit board that contains a ring of 12 RGB LEDs, a microcontroller, and some senors to:
1. Display the current time. Where the red LED represents the hour hand, the green LED is the minute hand and the blue LED is the second hand.
2. Display the current ambient temperature using the number of LEDs lit and the color of the LEDs for the temperature.
3. A magnetic compass with the red LED pointing north, and the blue LED pointing south
4. A read out of the current battery charge using the LEDs
5. A built in step counter
5. A flashlight that uses the red LEDs to create a red-light flashlight
6. Rechargable LiPO battery using USB-C
